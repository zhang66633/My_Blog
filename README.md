# My Tech Blog / 个人技术博客

基于 Tailwind CSS 的单页个人技术博客，展示项目、技术文章与联系方式。

**🌐 在线访问**：- https://zhang66633.github.io/My_Blog/
                 - https://blog-52h.pages.dev/
## 技术栈

- **Tailwind CSS** (CDN) — 原子化 CSS 样式
- **Google Fonts** — Space Grotesk（标题）+ DM Sans（正文）
- **原生 JavaScript** — 滚动动画、深色模式、移动端菜单、项目筛选
- **GitHub Pages** — 静态部署

## 页面结构

- 🏠 **Hero** — 个人介绍与社交链接
- 🚀 **Projects** — 项目卡片展示
- 📝 **Blog** — 技术文章列表
- 💡 **About** — 关于我
- 📬 **Contact** — 联系方式与留言表单

## 本地预览

直接用浏览器打开 `index.html` 即可，无需构建步骤。

```bash
# 或者使用任意静态服务器
npx serve .
python3 -m http.server 8080
```

## 部署

推送到 `main` 分支后，GitHub Pages 自动部署。
